# Functions

### Basic function with no parameters:

```
func FunctionName() {
  // code to be executed
}
```

### Function with parameters:

```
func FunctionName(param1 type, param2 type) {
  // code to be executed
}
```

### Function returns:

```
func FunctionName(param1 type, param2 type) type {
  // code to be executed
  return output
}
```
