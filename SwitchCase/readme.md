# Switch case examples

### Single switch-case example:

```
switch expression {
    case x:
        //do something
    case y:
        //do something
    case z:
        //do something
    default:
        //do something
}
```

### Run example:

```
go run SwitchCase/single-case.go 
```

### Multi switch-case example:

### Run example:

```
switch expression {
    case a,b,c:
        //do something
    case d,e,f:
        //do something
    case g,h,i:
        //do something
    default:
        //do something
}
```

```
go run SwitchCase/multi-case.go 
```
